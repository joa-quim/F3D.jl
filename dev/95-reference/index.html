F3D.ensure_f3d — Method
ensure_f3d() -> String

Ensures the F3D library is available under src/lib/ inside this package. Returns the directory containing the F3D C API library.

  1. Check if library already exists in src/lib/ → return early
  2. Query GitHub API for the nightly asset URL
  3. Download the archive
  4. Extract it
  5. Clean up the archive
  6. Return the directory containing the library
